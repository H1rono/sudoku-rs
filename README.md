# sudoku-rs

[![Rust](https://github.com/H1rono/sudoku-rs/actions/workflows/rust.yml/badge.svg)](https://github.com/H1rono/sudoku-rs/actions/workflows/rust.yml)

SUDOKU solver written in rust

## Usage

with cargo (installed with rustup):

```bash
cargo run < in.txt
```

with nix flake:

```bash
nix run < in.txt
```
